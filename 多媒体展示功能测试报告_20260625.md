**代码实现**:
```javascript
function renderImageMessage(text, localId, msgId) {
    const displayText = escapeHtml(text.replace(/\s*\(local_id=\d+\)/, ''));
    return `
        <div class="wechat-media-wrapper">
            <span class="wechat-media-type-tag">🖼️ 图片</span>
            <div class="wechat-image-item" onclick="showImagePlaceholder(this, '${msgId}')">
                <div class="wechat-image-placeholder" title="${displayText}"></div>
            </div>
        </div>
    `;
}
```

### 5. 链接识别与渲染

**实现功能**:
- ✅ 自动识别文本中的 URL
- ✅ 可点击的链接卡片样式
- ✅ 显示链接图标（🔗）
- ✅ URL 过长自动截断（40字符）
- ✅ 新窗口打开（target="_blank"）
- ✅ 消息统计：71条包含链接的消息

**代码实现**:
```javascript
function renderLinks(text) {
    const urlRegex = /https?:\/\/[^\s<>"]]+/gi;
    return text.replace(urlRegex, function(url) {
        const displayText = url.length > 40 ? url.substring(0, 40) + '...' : url;
        return `<a href="${url}" class="wechat-link-item" target="_blank" ...>
            <span class="wechat-link-icon">🔗</span>
            <span class="wechat-link-text">${displayText}</span>
        </a>`;
    });
}
```
