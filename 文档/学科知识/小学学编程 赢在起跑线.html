PART 1

小学阶段为什么要学编程？

不是跟风，而是为孩子打开一扇逻辑思维和未来竞争力的大门。

🧠

锻炼逻辑思维

编程的核心是"拆解问题 → 设计步骤 → 验证结果"。这个思维训练对数学、物理等学科的学习都有显著提升。

🏆

竞赛升学加分

CSP-J/S 认证和 NOIP 获奖在小升初、初升高自主招生中含金量极高。重庆多所名校重视信息学特长生。

⏰

越早起步越从容

五年级开始学 C++，到初一正好两年。初中课业压力大，提前打基础才能在关键节点从容参赛。

🔧

培养解决问题的能力

编程训练孩子面对复杂问题时不慌张，学会分步骤、找规律、试错修正，这是一辈子受用的能力。

🤖

AI 时代的核心素养

未来无论从事什么行业，理解计算思维、会和机器"对话"将成为基本素养，就像今天人人都要会用电脑一样。

🎯

专注力与耐心

写程序容不得半点马虎——一个分号的遗漏就能让程序报错。编程天然地培养孩子的细心和耐心。

PART 2

为什么选 C++，而不是 Scratch？

Scratch 很好玩，但我们的目标是 CSP-J 竞赛。
方向选对了，少走两年弯路。

🧩

Scratch / 图形化编程

  • ✕拖拽积木块，不是真正的代码
  • ✕偏游戏化、兴趣启蒙，无法对接竞赛
  • ✕到初中仍需从零学 C++，浪费时间
  • ✕CSP-J/S、NOIP 等竞赛不接受图形化语言
  • △适合低年级（1-3 年级）培养兴趣
⚡

C++ 语言

  • ✓信息学竞赛唯一指定语言（CSP-J/S、NOIP）
  • ✓直接接触真实代码和算法逻辑
  • ✓训练纯文本编程思维，为高中/大学打基础
  • ✓初学阶段可先纸笔学算法，不急着上机
  • ✓五年级开始完全来得及，循序渐进即可

👇 孩子们学的第一段 C++ 代码，其实就是这么简洁：

// 我的第一个 C++ 程序 #include <iostream> using namespace std; int main() { int a, b; cin >> a >> b; // 输入两个数 cout << a + b << endl; // 输出它们的和 return 0; }
PART 3

学习进度规划：五年级 → 初一

两年四阶段，每一步都踩在点上。目标明确：初一参加 CSP-J 并通过。

🌱 阶段一 · 五年级下学期（3-4 个月）

C++ 基础入门 · 理论先行

🎯 目标：掌握 C++ 语法基础，能手写简单程序的伪代码

📚 内容：

  • 计算机基础：二进制、变量、数据类型
  • 输入输出：cin / cout
  • 运算符与表达式
  • 条件语句：if / else / switch
  • 循环结构：for / while
  • 一维数组基础

💡 教学方式：板书 + PPT 讲解为主，不要求上机。重点训练学生用纸笔画流程图、写伪代码。

📅 频率：每周 2 次课，每次 1.5 小时

🌿 阶段二 · 六年级上学期（4 个月）

算法启蒙 · 开始上机

🎯 目标：掌握基础算法，能在洛谷完成入门题

📚 内容：

  • 函数定义与调用
  • 递归入门（斐波那契、阶乘）
  • 排序算法：冒泡排序、选择排序、插入排序
  • 查找算法：顺序查找、二分查找
  • 贪心算法初步
  • 字符串处理基础

💡 教学方式：理论与上机各占一半。开始在洛谷（Luogu）注册账号刷题。

📅 频率：每周 2 次课，每次 2 小时；课后每周 3-5 道在线练习

🔥 阶段三 · 六年级下学期（4 个月）

数据结构与进阶算法

🎯 目标：掌握 CSP-J 第二轮常考的数据结构和算法

📚 内容：

  • 栈（Stack）和队列（Queue）
  • 链表基础
  • 深度优先搜索（DFS）——回溯法
  • 广度优先搜索（BFS）——最短路径
  • 动态规划入门：0/1 背包问题
  • 树与图的基本概念和表示
  • 简单数论：素数筛、最大公约数（GCD）

💡 教学方式：以上机实战为主，配合 CSP-J 真题训练。

📅 频率：每周 2-3 次课，每次 2 小时；每周刷题 5-8 道

🚀 阶段四 · 小升初暑假（2 个月）· 冲刺！

全面冲刺 CSP-J

🎯 目标：通过 CSP-J 第一轮 + 第二轮，拿到入门级认证

📚 安排：

  • 第 1-2 周：第一轮笔试专项（计算机基础、复杂度分析、逻辑推理）
  • 第 3-4 周：第二轮上机真题专练（4 题 / 3.5 小时模拟）
  • 第 5-6 周：全真模拟考试 + 错题回顾 + 薄弱点突破
  • 第 7-8 周：查漏补缺 + 考前心态调整 + 冲刺训练

📅 频率：每周 4-5 次课，每次 2-2.5 小时；配合每日练习

🎯 备注：CSP-J 一般在每年 9-10 月举行，请关注 NOI 官网报名通知。

📝 第一轮（笔试）说明

  • 时间：2 小时，共 40 道选择题（含判断题）
  • 内容：计算机常识 + C++ 语法 + 算法基础 + 逻辑推理
  • 通过率相对较高，但不能掉以轻心
  • 建议用历年真题反复练习，熟悉题型

💻 第二轮（上机）说明

  • 时间：3.5 小时，共 4 道编程题
  • 使用 C++ 语言，需在 OJ 上提交代码
  • 部分分机制：即使拿不到满分，写出部分逻辑也有分
  • 重点训练：DFS/BFS、动态规划、模拟题
PART 5

给重庆家长的贴心建议

方向对了，方法对了，剩下的就是坚持。以下几点请一定记住：

1

重视数学基础，编程 ≠ 敲键盘

信息学竞赛的本质是算法和数学思维，不是比谁打字快。小学阶段同步加强数学训练（特别是逻辑推理、数论基础），比多刷 100 道编程题更有效。

2

选对老师和机构

重庆有多个信奥培训机构，建议选择有CSP/NOI 辅导获奖经验的老师。好老师不只是教语法，更能引导思维、把控节奏。可以先试听再决定。

3

每天 30 分钟 > 周末突击 5 小时

编程思维需要持续浸泡。每天花 30-40 分钟看书或做一道小题，远比周末集中刷 5 小时效果好。贵在坚持，不在时长。

4

善用在线评测平台

注册洛谷（Luogu）账号，从入门题开始刷。洛谷有详细的题解和讨论区，是全国信奥学生都在用的平台。养成"提交 → 看结果 → 改进"的习惯。

5

积累编程英语词汇

C++ 关键字、编译器报错信息、题目描述中都有英文。小学阶段同步积累 100-200 个常用编程英语词汇，学起来会轻松很多。

6

家长心态：重能力，轻功利

学编程最大的收获是思维方式的提升，证书和奖项是水到渠成的事。不要给孩子太大压力，保持兴趣和信心比什么都重要。

7

找同伴一起学

有小伙伴一起讨论问题，效果远好于独自摸索。可以加入学校的信息学兴趣小组，或者在洛谷上找同龄人互相交流。

8

注意保护视力

编程学习不可避免地用到屏幕，请遵循 20-20-20 法则（每 20 分钟看 20 英尺外 20 秒）。低年级阶段多用纸笔学算法，也是对眼睛的保护。

FAQ

家长常问的问题

🤔 孩子五年级了，零基础，现在开始还来得及吗？ +
完全来得及！五年级是公认的信奥起步黄金期。大部分 CSP-J 获奖选手都是五六年级或初一开始学的。按照本规划稳步推进，初一参赛是现实可行的目标。关键是现在就行动，而不是等到初中再后悔。
🤔 不学 Scratch，孩子能直接学 C++ 吗？会不会太难？ +
可以直接学！C++ 的入门语法其实并不复杂，变量、条件、循环这些概念和数学思维很接近。我们的规划特意安排了阶段一纯理论教学——先用纸笔和流程图理解逻辑，不急着上机操作，降低了入门门槛。五年级的孩子完全能理解这些内容。
🤔 学编程会不会影响学校主课成绩？ +
恰恰相反——编程训练的逻辑思维和问题分解能力对数学、理科成绩有正向促进作用。前提是要合理安排时间，按规划每周 3-4 小时的学习量不会影响主课，反而能让孩子做事更有条理。
🤔 CSP-J 考过了有什么用？ +
CSP-J 认证是由中国计算机学会（CCF）主办的全国性认证，含金量很高。通过后可以继续挑战 CSP-S（提高级）和 NOIP（省级联赛）。在重庆的重点中学自主招生中，CSP/NOIP 奖项是非常有竞争力的加分项。更重要的是，这段经历培养出的思维能力将受益终生。
🤔 需要买什么电脑和软件？ +
非常简单！一台普通笔记本电脑即可。软件方面，安装免费的 Dev-C++（Windows）或 VS Code（跨平台）作为编程环境。在线刷题用浏览器打开洛谷网站就行，不需要任何付费软件。前期理论阶段甚至不需要电脑。
