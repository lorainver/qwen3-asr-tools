核心解题方法论

以下七种方法贯穿本专题全部题目，理解方法比记住答案更重要。

method 01

方程法

万能方法。抓住"比较句"列等式，问什么设什么，写不下去时用未知数过渡。

method 02

合干分享 / 分干合享

合作完成的工程可拆分为各人独立完成的份额；分开做的工程可合并看待总量。

method 03

假设法

从鸡兔同笼演变而来。假设某一条件改变，用"多出"或"不足"的部分求解。

method 04

比例法

时间相同时，工作量与效率成正比。已知比值与总和即可求各量。

method 05

量率对应

找到具体数量占整体的几分之几，用"具体量 ÷ 对应分率 = 整体"求解。

method 06

周期分析法

轮流做工本质是周期问题：求每周期工作量 → 求整周期数 → 求零头。

method 07

关键句翻译

数学的本质是翻译。将文字语言（比较句、陈述句）转化为数学表达式。

一、工程问题

工程问题的核心公式：工作量 = 效率 × 时间，即 t = S ÷ v。以下五道题覆盖工程问题的主要题型。

工程 01

合作者中途休息

甲单独完成一项工程需 40 天，乙单独完成需 60 天。两人合作若干天后，甲因病休息，乙继续工作。从开工到完工共用了 27 天。问：甲休息了几天？
1

合干分享法（算术）

思路：乙全程工作 27 天，先算出乙完成的工作量，再用剩余工作量除以甲的效率得甲的工作天数。

// 乙 27 天完成的工作量 乙完成 = 27 × 1/60 = 27/60 = 9/20 // 甲完成的工作量 甲完成 = 1 − 9/20 = 11/20 // 甲工作的天数 甲天数 = (11/20) ÷ (1/40) = 22 天 // 甲休息的天数 休息 = 27 − 22 = 5 天
2

假设法 — 直接求休息天数

假设甲不休息，两人合干 27 天，总工作量会超过"1"。多出的部分正是甲休息那几天本应做的工作。

// 合效率 v合 = 1/40 + 1/60 = 1/24 // 假设不休息的总工作量 总量 = 27 × 1/24 = 27/24 = 9/8 // 超出部分 = 甲休息期间应做的量 超出 = 9/8 − 1 = 1/8 休息天数 = (1/8) ÷ (1/40) = 5 天
3

假设法 — 通过正常完工天数求解

先算甲不休息时两人合干的天数，再通过乙多干的天数反推甲少干的天数。

// 正常合干完工天数 正常天数 = 1 ÷ 1/24 = 24 天 // 乙比正常多干了 3 天 乙多干 = 3 × 1/60 = 1/20 // 甲比正常少干的天数 甲少干 = (1/20) ÷ (1/40) = 2 天 // 甲实际工作 甲工作 = 24 − 2 = 22 天 休息 = 27 − 22 = 5 天
合干分享的关键在于理解：总工作量 = 甲完成量 + 乙完成量。当题目告知"分干"（一人休息），即可用"合享"（总量减去另一人的份额）来求解。假设法的精髓是：改变一个条件，用差值反推原问题。
工程 02

多人合作求整体效率

五人 A、B、C、D、E 合作完成一项工程。已知：
A、B、C 合作需 7.5 小时；A、C、E 合作需 5 小时；
A、C、D 合作需 6 小时；B、D、E 合作需 4 小时。
问：五人合作需几小时完成？
1

翻译条件 + 整体叠加法

将每个条件翻译为效率方程，再通过巧妙叠加求五人效率之和。

// 翻译四组条件为方程 ① A + B + C = 1/7.5 = 2/15 ② A + C + E = 1/5 ③ A + C + D = 1/6 ④ B + D + E = 1/4

观察各字母出现次数：

字母出现方程出现次数
A① ② ③3 次
B① ④2 次
C① ② ③3 次
D③ ④2 次
E② ④2 次

B、D、E 各少一次，恰好在方程④中同时出现，因此将④多加一遍：

// ① + ② + ③ + 2 × ④，每个字母恰好出现 3 次 3(A+B+C+D+E) = 2/15 + 1/5 + 1/6 + 2 × 1/4 = 2/15 + 3/15 + 1/6 + 1/2 = 1/3 + 1/6 + 1/2 = 1 A + B + C + D + E = 1/3 // 五人合干时间 t = 1 ÷ (1/3) = 3 小时
多人多条件题型的通用策略：不必逐一求出每个人的效率。只需将方程巧妙叠加，使每个未知数出现相同次数，从而直接求出总和。
关键技巧：统计每个字母出现次数，找到能"补齐"的方程多加一遍。
工程 03 · 经典

三人两库

甲、乙、丙三人同时开始搬运两个相同仓库的货物。甲每小时搬 1/6 个仓库，乙每小时搬 1/7 个仓库，丙每小时搬 1/14 个仓库。丙先帮甲搬运，后又去帮乙搬运，最后两个仓库同时搬完。
问：丙帮甲多少小时？帮乙多少小时？
1

传统方法 — 先求总时间

// 三人合效率 v合 = 1/6 + 1/7 + 1/14 = 14/84 + 12/84 + 6/84 = 32/84 = 8/21 // 总工作量为 2（两个仓库） 总时间 t = 2 ÷ 8/21 = 21/4 小时 // 甲在这段时间内自己完成的量 甲完成 = 1/6 × 21/4 = 7/8 // 丙帮甲完成的量 丙帮甲量 = 1 − 7/8 = 1/8 丙帮甲时间 = (1/8) ÷ (1/14) = 7/4 小时 // 乙在这段时间内自己完成的量 乙完成 = 1/7 × 21/4 = 3/4 丙帮乙量 = 1 − 3/4 = 1/4 丙帮乙时间 = (1/4) ÷ (1/14) = 7/2 小时 // 验证：7/4 + 7/2 = 21/4 = 总时间 ✓
2

比例法 — 利用时间相同，工量比等于效率比

三人工作时间相同，因此工作量之比等于效率之比。

// 效率通分：乘以 42 甲 : 乙 : 丙 = 1/6 : 1/7 : 1/14 = 7 : 6 : 3 // 总份数 = 16，总工作量 = 2 每份 = 2/16 = 1/8 甲完成 = 7 × 1/8 = 7/8 → 丙帮甲 = 1/8 乙完成 = 6 × 1/8 = 3/4 → 丙帮乙 = 1/4 丙帮甲时间 = (1/8) ÷ (1/14) = 7/4 小时 丙帮乙时间 = (1/4) ÷ (1/14) = 7/2 小时
三人两库是高频经典题。核心突破口：三人工作时间相同 → 工量比 = 效率比。知道比值和总量，即可用"按比分配"快速求解，无需先算总时间。
进阶：如果两个仓库大小不同（如 A 是 B 的 1.25 倍），需先用分数表示两个仓库的大小关系，再用同样思路求解。
工程 04

人员分派问题

一批工人分配到甲、乙两个工地。甲工地的工作量是乙工地的 1.5 倍。
上午，去甲工地的人数是去乙工地人数的 3 倍；下午，这批工人的 7/12 去了甲工地。
到傍晚时，甲工地的工作已完成，乙工地还需 4 名工人再做一天才能完成。
问：这批工人共有多少人？
1

方程法 — 设人数列等式

设总人数为 12x（取 12 的倍数便于处理 1/4 和 7/12），每人半天做 1 份工作。

// 人员分配 上午：甲工地 3/4 × 12x = 9x 人，乙工地 1/4 × 12x = 3x 人 下午：甲工地 7/12 × 12x = 7x 人，乙工地 5/12 × 12x = 5x 人 // 工作量（每人半天 = 1 份） 甲工地 = 9x + 7x = 16x 乙工地 = 3x + 5x + 4×2 = 8x + 8 ← 4人再做1天=8份 // 列方程：甲 = 1.5 × 乙 16x = 1.5 × (8x + 8) 16x = 12x + 12 4x = 12 x = 3 → 总人数 = 12 × 3 = 36 人
2

量率对应法

将全部工人一天的工作量设为"1"，用分率表达各工地完成量，再用"具体量 ÷ 分率 = 整体"求解。

// 每半天 = 1/2 甲当天完成：3/4 × 1/2 + 7/12 × 1/2 = 3/8 + 7/24 = 2/3 乙当天完成：1/4 × 1/2 + 5/12 × 1/2 = 1/8 + 5/24 = 1/3 // 甲 = 1.5 × 乙总量 → 乙总量 乙总量 = 2/3 ÷ 3/2 = 4/9 // 乙剩余工作量 乙剩余 = 4/9 − 1/3 = 1/9 // 4 名工人做 1 天 = 总工人 1 天工作量的 1/9 4 人 = 全体的 1/9 → 总人数 = 4 ÷ 1/9 = 36 人
方程法是"万能保底"方法——不确定怎么做的时候，先找等式、列方程。量率对应则更加巧妙，将"4 名工人做 1 天"与全体的关系用分率表达出来。两种方法殊途同归，都值得掌握。
工程 05

轮流做工（周期问题）

一项工程，甲单独做需 6 小时完成，乙单独做需 8 小时完成。现由甲、乙两人轮流各做 1 小时（先甲后乙）。问：共需多少小时完成？
1

周期分析法

轮流做工本质是周期问题：每个周期 2 小时（甲 1 小时 + 乙 1 小时），按"求周期 → 求整周期数 → 求零头"三步走。

// 第一步：求每周期工作量 每周期 = 1/6 + 1/8 = 4/24 + 3/24 = 7/24 // 第二步：求整周期数 整周期数 = 1 ÷ 7/24 = 24/7 ≈ 3 个整周期 3 个周期完成：3 × 7/24 = 21/24 = 7/8 耗时：3 × 2 = 6 小时 // 第三步：求零头 剩余 = 1 − 7/8 = 1/8 1/8 < 1/6（甲的效率），由甲完成 零头时间 = (1/8) ÷ (1/6) = 6/8 = 3/4 小时 总时间 = 6 + 3/4 = 6¾ 小时
轮流做工三步法：①算每周期工作量；②算整周期个数和剩余量；③按顺序分配零头。
类似模型：蜗牛爬井、进退交替等，底层逻辑相同——周期分析 + 尾部处理。务必先掌握基础周期问题，再挑战复杂轮流题。

二、配套问题

配套问题的核心：找到配比关系（关键句），将其转化为等式。

配套 01

盒身盒底配套

用铁皮做罐头盒。一张铁皮可制盒身 15 个，或盒底 45 个。一个盒身与两个盒底配成一套罐头盒。现有一百五十张铁皮，用多少张制盒身、多少张制盒底，可以正好制成配套的罐头盒？
1

方程法 — 翻译配比关系

关键句：一个盒身与两个盒底配成一套。翻译为数学语言：盒底数 = 2 × 盒身数

// 设 x 张铁皮制盒身，则 (150 − x) 张制盒底 盒身数 = 15x 盒底数 = 45(150 − x) // 配套关系：盒底 = 2 × 盒身 45(150 − x) = 2 × 15x 6750 − 45x = 30x 6750 = 75x x = 90 盒身用 90 张，盒底用 60 张 // 验证：盒身 15×90 = 1350，盒底 45×60 = 2700，2700 = 2×1350 ✓
配套问题解题三步：①找到关键句（配比关系）→ ②翻译为等式 → ③设未知数求解。
分析句子类型：陈述句（描述过程）、比较句（列等式的关键）、问句（目标）。重点勾画比较句，而非所有带数字的句子。

学习总结与建议

解题思维框架

  1. 读题翻译：将文字语言转化为数学表达式。分清陈述句（描述过程）和比较句（列等式）。
  2. 分析目标：问什么？需要什么条件？从问题出发逆向推理。
  3. 选择方法：方程法（万能保底）、算术法（更简洁灵活）、比例/假设法（特定场景）。
  4. 一题多解：用一种方法做完后，尝试其他方法。做对不是终点，找到最优解才是。
  5. 分类总结：给题目"贴标签"——三人两库、轮流做工、配套问题……分类越清晰，考试时提取越快速。

给家长的建议

  1. 引导孩子找关键句：每道题都让孩子指出"哪句话可以列等式"，养成精准审题习惯。
  2. 鼓励讲解思路：能做对但说不清为什么的孩子，需要加强"表达"训练——讲得清才是真懂。
  3. 简单题练透：一道简单题用三种方法做，胜过三道难题各做一遍。深度优于广度。
  4. 接受重复学习：好的题目值得做两三遍，每遍可能有新的发现和更优的解法。
