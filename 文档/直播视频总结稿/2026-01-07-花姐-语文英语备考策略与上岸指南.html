第一章

直播间定位与目标受众

本直播间属于小众定向输出，核心围绕优生规划展开，不适用于追求"快乐教育"或完全认同阳光招生政策的家长群体。

面向小学阶段

  • 学有余力的小学优生（小四、小五、小六）
  • 重点小学中排名靠前的学生
  • 普通小学中班级前几名的学生

面向初中阶段

  • 已进入青龙珠校实验班以上、年级排名前100名左右的A类生
  • 区重学校（如二十九中、求精、复旦）年级前100名
  • 两八、科八、鲁巴等校实验班以上学生
核心定位：针对A类生和B+生（三门主课中有一门偶尔跌至B级、有提升可能的学生）。B类及以下学生建议优先走区内推优路线，夯实校内基础。
第二章

优生的定义与分层标准

层级标准描述建议路径
A类生 重点小学班级前几名；普通小学班级前3~5名；语数均在95分以上 冲四大金刚/七龙珠实验班以上，走竞赛或二加四通道
B+生 三门主课中有一门偶尔B级，其余均在A线以上 可尝试竞赛或二加四，同时保持综合成绩
B类生 综合成绩中等，某一科目有明显短板 针对性补弱（如语文刷真题），争取区内推优
B-及以下 校内成绩不突出，班级排名靠后 优先夯实校内知识，走区内推优，期望值不宜过高
关键提醒：区内推优要求成绩在区内排名前50%以内才有机会。若连前50%都未达到，区里推优基本无望，更不适合走竞赛赛道。
第三章

五次衔接机会全景图

与巴蜀本部高中（八本）衔接的核心路径共五次机会（加上小学阶段共称"五加一次"），其他四大金刚及龙珠校基本跟随八本节奏，时间上提前一到两个月。

第零次 · 小学阶段（零加六）

小四至小六期间，通过数学竞赛或信息竞赛与本部高中建立联系。最早四年级上岸，五年级大规模选拔。入选者几乎跳过正常初中学习，直接与高中竞赛板块无缝衔接（即"零加六"）。选拔科目：数学竞赛、信息竞赛、美术方向。

第一次 · 初一月考后（物化生竞赛选拔）

初一入学后第一次月考即启动选拔。从全校4000~5000人中选出前1200名参加数学AB卷选拔（A/B卷各10道难题），最终选出约300人进入物化生竞赛培优。数学和信息仍延续小学阶段的童子功训练。

第二次 · 初二十月（物化生二次筛选）

开学后对物化生竞赛进行第二次筛选，部分学生被淘汰或主动退出，同时综合特优生也可能被纳入选拔。数学AB卷难度继续提升。

第三次 · 初二寒假（竞赛二加四 / 学堂一班组建）最关键

这是最重要的选拔节点。针对所有竞赛生（数学、物理、化学、信息、生物五大学科）及综合特优生，选拔学堂一班（竞赛二加四）。考核科目：数学和物理的AB卷各三小时。最终学堂一班规模约120~150人（分A/B班），外校综合优生也可递交简历参选。同时可开始接收第二高中本部名单。

第四次 · 初二暑假（综合二加四）

选拔综合二加四（综合学堂班），考核四门科目：数学和物理AB卷各三小时，语文和英语各一小时。组建综合学堂班，每班约50人，三个班（可能扩至四个班），为未来冲刺C9/华五储备力量。

第五次 · 初三三月（保送考试）

初三三月份的保送考试，考核七门科目（与中考高度关联）。若前四次机会均未抓住，这是最后的自招窗口。保送率按AB档计算约80%，含CD档可达90%以上。中考结束后还会有估分群提供降分约和套餐约谈判支持。

警示：整个初中阶段A类生约700人线，二加四通道提供约300人名额（每校约150人 × 7所学校 ≈ 2100人）。若拖至保送阶段，名额大幅缩减且需应对七门考试，性价比显著下降。
第四章

小学阶段：零加六与竞赛启蒙

小学阶段对应"五次机会"中的零加六。核心逻辑：小学优生通过竞赛方向（数学/信息）或美术方向与名校建立早期链接。

4.1 小学阶段三件核心事项

  1. 竞赛方向一：数学竞赛 — 系统学习小奥基础后进入竞赛板块
  2. 竞赛方向二：信息竞赛 — 编程方向，需评估孩子综合成绩是否能支撑
  3. 美术方向：若孩子有美术天赋，走美育路线（三年约10万元），可衔接名校美育项目

4.2 不学奥数的出路

客观说明：若孩子属于学困型（校内90分以下都困难），应优先抓校内成绩，争取区内推优。所有直属校不进行区内推优，均需通过内部选拔。不学奥数 = 无法参加选拔 = 天花板仅限龙珠校分校。

4.3 上岸后的关键动作

  • 已上岸但学校可能暂停早培的，必须去参加高联一试免费测试，判断孩子是否适合竞赛方向
  • 小四、小五、小六阶段，快速完成小奥后立即进入竞赛方向或高阶早培
  • 无论上岸与否，校内语文和数学均不得低于95分
核心观点：只要有机会、有小范围隐蔽的活动出现，奥数仍然是筛选竞赛思维最优手段。学有余力的优生，奥数一定值得学。学完基础后进入竞赛板块，比拼的是思维深度而非刷题量。
第五章

初一：首次选拔与竞赛分流

初一第一次月考即启动选拔流程，节奏显著提前（以往初二才开始，现已提前至初一国庆期间）。

5.1 初一月考选拔

  • 时间：九月入学 → 十月第一次月考 → 十一期间启动选拔
  • 规模：从4000~5000名大八系学生中，通知前1200名参加选拔
  • 形式：数学A/B卷（10道难题）
  • 结果：选出约300人进入物化生竞赛培优（物理、化学、生物各约100人）
  • 数学和信息延续小学童子功训练
重要说明：学校老师口头说"有兴趣的都可以去选拔"，实为体现公平性的客气话。实际通知的是前120名左右，排名靠后的即使参加也很难入选。

5.2 竞赛身份的价值

  • 竞赛不是"要么拿金牌要么没用"，而是多次衔接机会的入场券
  • 有竞赛身份的学生，在各选拔环节会获得更多机会
  • 竞赛锤炼后的学生，高中综合成绩往往非常突出
  • 即使最终成为"炮灰"，竞赛训练对数学思维和抗压能力的提升依然有长期价值
正面结论：学竞赛的孩子，退赛后回转综合方向，往往能冲到年级前50名。竞赛科目（除信息外）均与高考科目关联，不会浪费时间。
第六章

初二：核心选拔与二加四组建

6.1 初二十月：物化生二次筛选

  • 对初一已入选的竞赛生进行二次筛选
  • 部分学生自然淘汰（主动或被动），部分综合特优生可参与
  • 竞赛班可能分A/B队，进度和难度有差异
  • 数学AB卷难度继续提升

6.2 初二寒假：最关键选拔——学堂一班

这是整个五次机会中最核心的节点。

维度详情
参选范围所有竞赛生（数学/物理/化学/信息/生物）+ 综合特优生（需数学和物理排名达标）
考核科目数学AB卷各3小时 + 物理AB卷各3小时
录取规模学堂一班约120~150人（分A/B班），综合学堂班约150人（3个班，每班50人）
外校参选外校综合优生可递交简历参选，可能获得更简单的题目
保送名额学校力保，但不会明确承诺；实际给予的概率极高
关键警示：学堂一班的选拔，B卷才是拉开差距的关键。A卷基于校内进度，而B卷涉及竞赛方向题目 + 高中内容。未提前准备B卷的学生，很可能"每个字都认识但不会做"。

6.3 初二暑假：综合二加四

  • 正式组建综合学堂班，每班约50人
  • 考核四门：数学AB卷3小时 + 物理AB卷3小时 + 语文1小时 + 英语1小时
  • 综合学堂班为未来冲刺C9/华五的核心战队
  • 保送名额在此阶段确定
第七章

初三：保送考试与最后机会

初三三月的保送考试是最后一次自招窗口。

7.1 保送考试要点

项目说明
考试科目七门（与中考高度关联）
难度定位综合压轴难度
保送率AB档约80%，含CD档达90%以上
名额特点不一定能给到三位数；相比二加四，名额显著减少
体系类时间巴蜀体系类（柯巴、贵巴、黄巴、虎巴等）等三月份

7.2 中考后的估分谈判

  • 中考结束后，会有估分群提供降分约和套餐约谈判支持
  • 学校可能给出5~10分降分约
  • 套餐约：第一志愿填八本/第二高中本部，第二梯队填科八/两八高中，第三梯队填B八高中
  • 第二高中本部已开始收名单，名额有限，需尽快行动
核心建议：不要等到保送阶段才行动。保送考七门，名额少，压力远大于二加四（仅四门）。提前上岸后，重心应转向初高衔接，而非继续卷中考。
第八章

AB卷考试深度剖析

AB卷是巴蜀系选拔的核心形式，A卷与B卷的定位和难度有本质差异。

8.1 A卷定位

  • 基于校内学习进度，但以八本的教学目标为准
  • 难度高于普通区中约20%，高于四大金刚约10%
  • 考察本学期内容的深化与综合应用

8.2 B卷构成

B卷考核三大内容板块：

  1. 压轴升阶：本期学校内容的深化压轴题（A卷已部分涉及）
  2. 竞赛方向：初联级别的数论、几何、代数、组合等题目
  3. 高中内容：高中知识点的前半段（高一相关内容）
关键提醒：最终得分差距主要体现在B卷上。A卷因校内选拔的持续训练，差异不大；而B卷是否学过，差距可达10~20分甚至更多。必须通过针对性训练攻克B卷。

第九章

各学科培养策略

数学

  • 属于童子功，小学阶段必须系统学习
  • 初一即进入高联一试级别训练
  • 高中必修体系分四期，每期15课时，共60课时
  • 数学是所有选拔的核心科目，AB卷重心所在

物理

  • 初一刚开课，可通过竞赛培优半道插入
  • 初中物理必修体系分四期
  • 与数学并列为二加四选拔的两大核心科目
  • 高中阶段建议至少跟两轮（校外一轮 + 校内一轮）

化学

  • 初三才开课，初中仅两个学期内容
  • 可找好老师快速完成一个学期的课程
  • 难度上提升空间有限，不必过度投入
  • 初中化学必修约20多个课时

信息竞赛

  • 与数学思维关联度高，但与高考科目不直接相关
  • 最大风险：综合成绩容易掉队、游戏瘾难控制
  • 退赛后回转综合最不容易追上
  • 每年省队约8~9人，淘汰率极高

英语

  • 唯一可以在小学阶段提前完成的学科
  • 建议从小学一年级开始原版阅读（如小爱英语）
  • 单词包（约2980元/年）可覆盖初中+高中全部词汇
  • 每天15分钟背单词，一年时间解决70%以上英语问题
  • 初中阶段背完高中单词，英语可稳定在140分以上

语文

  • 理科优生普遍的薄弱环节
  • 可通过刷中考真题提升（300客观 + 300主观）
  • 语文是有套路的：找题眼、字眼，出题逻辑有迹可循
  • 目标：中考语文从110~120分冲至125~130分
  • 每周末线上2小时 + 寒假10天 + 暑假15~20天即可见效
警示：信息竞赛学生退赛后，综合成绩最不容易恢复。所有竞赛退赛学生中，信息竞赛退赛者的语文和英语往往问题最大。务必在竞赛期间保持英语每日阅读和单词背诵。
第十章

高阶早培与初高衔接

高阶早培和初高衔接是与八本产生实质链接的核心手段，也是从竞赛/二加四上岸后最关键的后续动作。

10.1 高阶早培

  • 必须使用巴蜀自编教材（初中版叫《曙光》，高中版叫《智库》）
  • 按板块学习（代数 → 函数 → 后续板块），难度大于区中约20%
  • 数学分六期，约一年至一年半完成
  • 物理分四期
  • 化学约20多个课时
  • 核心目的：为第一次月考后的选拔做准备
核心策略：高阶早培 ≠ 外面机构快速翻书。必须找有坑校背景、使用巴蜀教材体系的小班精品教学。进度要跟校内初一初二同步，认认真真学，而非一个假期翻两期。

10.2 初高衔接（强基版本）

二加四上岸或保送上岸后，必须规划初高衔接，原因如下：

  • 高中难度是初中的9~10倍，数学一个学期涉及200多个知识点，60~70%集中在高一
  • 高中分科后第一次排名决定一清班/二清班归属
  • 高一下学期期末考试决定年级前100名的定位

强基版本课程结构（五三二阵型）

板块占比内容
校内同步中档50%高中必修体系的基础知识与中档题
压轴30%高中校内压轴题训练
强基20%强基板块内容（数学四期 + 物理四期）
关键提醒：高中至少要跟两轮强基课程（一轮校外 + 一轮校内），每轮4期 × 15课时 = 60课时。必须避开大班教学，选择能叫出孩子名字的小班精品老师。
第十一章

选课与机构选择建议

推荐选择

  • 小班精品教学（10人左右），有坑校/校队背景
  • 使用巴蜀体系自编教材
  • 老师能密切跟踪学校培优节奏
  • 有真题题库和多届选拔经验
  • 价格略高但含金量有保障

不推荐选择

  • 大机构的二三十人大班（为B/C类生服务）
  • 一个班三四十人、"拍屁股走人"式教学
  • 以中考660~670分当优秀案例展示的机构
  • 进度快但不扎实的翻书式培训
  • 离目标学校太远、不了解该校选拔体系的机构
补充说明：优生的加餐应是"牛排级"的精品细料，而非大杂烩。靠近哪所学校就学哪所学校的体系和经验。如果目标是八本，就应该在八本附近找有巴蜀教研体系的培训机构。

第十二章

各年级行动清单

小三~小四

  1. 以最快速度完成小奥基础学习（约一年时间）
  2. 英语启动原版阅读（小爱或同类产品），每天30分钟
  3. 评估是否适合信息竞赛方向
  4. 小四开始短暂刷题，规划一门竞赛方向
  5. 语文保持阅读积累

小五~小六

  1. 已上岸：立即进入高阶早培（巴蜀体系教材），数学和物理优先
  2. 未上岸：争取上岸机会，同时不放弃高阶早培
  3. 英语：开始背单词（单词包），每日15分钟
  4. 参加高联一试免费测试，判断竞赛适配度
  5. 校内语文和数学不低于95分
  6. 美术方向：三年投入约10万，可走清北美育

初一

  1. 第一次月考后参加数学A/B卷选拔
  2. 入选竞赛培优后，物化生三科启动系统训练
  3. 数学和信息延续童子功训练
  4. 英语：初中阶段必须背完高中单词（每天15分钟）
  5. 同步进行高阶早培（巴蜀体系A卷内容）
  6. 若语文薄弱，启动中考真题刷题计划

初二

  1. 十月：参加物化生二次筛选
  2. 寒假：冲刺学堂一班选拔（数学+物理AB卷各3小时）
  3. 二加四冲刺：数学60+课时 + 物理相应课时
  4. 若竞赛二加四未上岸，暑假冲刺综合二加四
  5. 高阶早培进入B卷阶段（竞赛方向 + 高中内容）
  6. 综合成绩密切关注，确保不低于A线

初三

  1. 三月：保送考试（七门科目）
  2. 已获死约的学生：重心转向初高衔接，不再纠缠班型
  3. 未上岸的学生：全力准备中考保送
  4. 中考后：进入估分群，争取降分约和套餐约
  5. 高中入学后：紧跟强基版本初高衔接（五三二阵型）

高中阶段

  1. 高一上学期期末 + 半期考试决定分科排名
  2. 年级前100名进入一清班，前250名进入强基系列
  3. 强基课程至少跟两轮（校外 + 校内各一轮）
  4. 竞赛生高一结束前决定是否退赛，不要留恋
  5. 目标：C9/华五为主流，清北为冲刺方向
保障信息：只要在五次衔接机会中抓住任何一次，都有可能进入最好的班型。即使成为竞赛"炮灰"，综合成绩依然优秀、学习能力突出、抗压能力强劲，学校不会让这样的孩子掉队。
