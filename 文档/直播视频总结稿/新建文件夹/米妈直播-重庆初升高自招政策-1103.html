核心结论

02

招生格局：自招名额与套餐约机制

自招名额充裕的学校类型

以下几类学校自招名额相对充裕，初升高阶段值得重点关注：

  • 红八（巴蜀本部）：自招名额为四大金刚中最多，约为其余三校的三倍，所有招生均为自招名额，不参加联招考试。
  • 南渝、两江八中、两江巴蜀：均无联招代码，以自招为主，招生名额较同等水平学校更为充裕。
  • 区内集团校含普高名额的学校：区重点或龙珠校集团旗下拥有普通高中学籍的学校，可通过套餐约实现"兜底"。
  • 一中系：内签约六百余人，但正式"死约"仅一百至一百五十余份，大部分为套餐约，由天星桥中学等分校托底。

套餐约机制详解

套餐约的本质

学生签约时同时绑定本部与分校（或集团校）的录取条件。达到约定分数线则进入本部对应班型，未达到则自动分流至绑定学校。

  • 巴蜀体系：八本签套餐约 → 达黄花园线可获实验班 / 超线十分获清北班 → 未上线则无条件进入两江巴蜀清北班
  • 一中体系：签一双/一际套餐约 → 未达线则学籍落至天星桥中学，由天星托底
  • 朝阳体系：朝阳有自有普高托底，可给出三十分降分签约
普高托底的关键作用

联招线以下仍可填报志愿。部分区重点学校名下有普通高中，学籍挂靠普高但实际在区重点就读，形成"低分兜底"机制。这一模式正成为各校破解5%自招名额限制的核心策略。

03

八中体系：布局分析与困境

当前格局

  • 红八高中：仅招六个班（其中两个班为2+4直升），名额极为有限，红八本校学生竞争激烈，外部学生基本无法进入。
  • 科学城八中：定位为原艺术高中，中考录取仅630余分。与红八高中分差达55至60分，导致套餐约吸引力不足。
  • 两江八中：高中刚办第二届，尚在起步阶段。
  • 张八（张家花园八中）：今年起高中停办，师资将转移至两江巴蜀高中。未来将转型为单办初中。

核心困境

八中高中断档问题

八中体系长期"初中强、高中弱"，高中仅沙巴渝巴双本部，下接断档。相比一中系（一实、一际均为高完中）、三中系（南渝等均为高完中），八中的高中梯队布局明显不足。红八高中分差过大导致套餐约缺乏吸引力，牛娃及家长普遍不愿签约。

两江巴蜀的崛起

双本部定位 重点

两江巴蜀被定位为"双本部"，享有巴蜀系最高级别扶持。两江新区投入重金建设，巴蜀方面需派师资入驻。今年起由八本协助联合招生，全部为自招名额。张八高中停办后，原有师资将下沉至两江巴蜀高中。作为全新校区，家长对其"一张白纸"充满期待。

育才北部的被低估

育才的"没落"实质是招生策略调整，而非教学质量下降。其清北班师资实力扎实，含金量远超多数龙珠分校，属于严重被低估的学校。

05

自招比武题型变化

变化的根本驱动

新高考 → 新中考 → 自招命题连锁反应

重庆是最早实施新高考的地区之一。新高考的题型变革（如多选题）向中考传导，四大金刚的自招命题随之调整。自招考试本质上是学校选拔竞赛苗子和拔尖学生的工具，题型设计直接对接高联二试和高中衔接能力。

数学题型三大变化

填空题尾部 变化一

旧：翻折几何 + 不定方程
新：圆与几何问题 + 阅读材料题

原因：新中考加大材料阅读和计算题比重。

选择题尾部 变化二

旧：不等式、组合、反比例函数
新：翻折几何 + 代数多选题

原因：新高考已出现多选题型，自招率先引入。

压轴题核心转变 变化三 · 最关键

函数压轴：由特殊三四边形 → 角度的存在性问题
几何压轴：由线与线的关系 → 线与圆的关系（引圆问题）

引圆问题要求学生通过作辅助圆解题，这是初二培优及竞赛体系才涉及的内容，校内普通教学不涵盖。

引圆问题的本质

巴蜀和八中的自招几何题之所以"超级难"，核心在于引入了引圆问题（线圆关系），这属于高联二试的竞赛考点。能在自招中做出此类题目的学生，可直接签约竞赛班。巴蜀、八中体系内的培优学生在初二即接触此类题型，而多数龙珠校和非四大金刚体系的学生完全未涉及。

06

学科考点深度分析

数学

  • 引圆问题是四大金刚（尤其巴蜀、八中）的核心拉分点，需初二及以上水平
  • 多选题型已出现在自招卷中，中考尚无，自招先行
  • 培优生初二即可应对大部分题型；竞赛生初一即可尝试
  • 中考最后一题最后一问：可用高中三角恒等变换（两角和差公式）快速求解，但未超前学的学生基本无法完成

物理

  • 核心考点：力学整体法与隔离法——中考不考，但自招（尤其巴蜀、八中）必考
  • 自招比武中，力学比电学更重要，因其直接对接高中物理体系
  • 初中电学属于独立模块，高中物理的电学本质仍是力学
  • 非八中、巴蜀体系的学生普遍未接触整体法和隔离法，属于明显劣势
  • 初二力学板块是自招备考的关键节点

英语

  • 考查首字母填空和翻译题，题型接近上海新中考
  • 阅读难度与四大金刚校内水平持平或略高
  • 四大金刚学生应对相对从容，非四大学生可能感到吃力

语文

  • 诗词鉴赏——属于高考考点，中考不考，但四大金刚自招必考，体现高中衔接导向
  • 文言文——由高一教师出题，仅考实词翻译，难度显著高于初中水平
  • 已有八中学生反映考一中时语文（文言文和诗词鉴赏）最具不确定性
  • 非四大体系学生须提前熟悉此类题型，否则无从下手
核心启示：培优体系决定竞争力

自招卷并非"高中内容"，而是四大金刚校内培优体系的延伸。八中、巴蜀的培优教研远超普通龙珠校数个层级。未进入培优体系或非四大金刚体系的学生，在自招中面对引圆问题、力学整体法与隔离法、诗词鉴赏等考点时，将面临系统性劣势。自招中五到十分的分差即构成分水岭。

升学规划建议

关键时间节点决策

  • 初二阶段：核心分水岭。需确定走内迁保送、翻墙自招、指标到校、艺体生还是国际方向。力学和引圆问题的知识储备在此阶段奠定。
  • 初三上学期：参加比武见题型，明确目标学校层次，避免盲目报考造成打击。
  • 寒假期间：初二学生翻墙考/转学考的关键窗口，培优生可利用此阶段突破。
  • 志愿填报期：注意学校套路（如指标到校表格），所有关键决策需专业人士把关。

不同路径的策略

八中/巴蜀体系路径

理科培优扎实，自招竞争力强，初升高可拿高分。选择此体系需家长和孩子具备"越强则强"的抗压能力。高中出口需提前规划——红八高中名额极其有限，需同步考虑两江巴蜀、科学城八中等备选。

国际方向路径

最佳转入时间为初二升初三。需对孩子的就业方向、大学专业、综合能力进行全面评估后决策。理科优势在国际方向同样显著。建议在初升高时即权衡利弊，获取多所学校奖学金后综合选择。

区县/非联招区路径

信息差是最大障碍。需通过转学考进入四大金刚或龙珠体系，再参加翻墙考争取自招名额。建议尽早接触真题，了解真实难度，避免被机构伪题误导。

核心原则

避免信息差，理解政策表象背后的真实逻辑

四大金刚均为套餐约，但不同体系的套餐约吸引力差异巨大——核心在于分校与本部分差是否在可接受范围内。自招题型变化的根本原因是新高考改革的传导效应，而非简单的"考高中内容"。掌握这些底层逻辑，才能做出精准的升学决策。
