SECTION 01

考试概况与设计理念

试卷定位、板块分布、难度策略及分数线说明

核心定位 本卷为中阶选拔测试卷，目标是筛选适合进入中阶学习的学生。试卷覆盖几何、代数、数论、组合四大板块，考试时间 120 分钟。

线上与线下差异

  • 线上试卷与线下试卷不完全一致，部分题目有所调整
  • 线上供题老师更多，题目来源更广

难度设计思路

  • 各板块均设两道题，允许学生在擅长板块多做
  • 整体难度适当上调，保证优秀学生不被遗漏
  • 填空题部分考察一试内容
代数板块特殊性 代数板块进度要求最快——需要学完整个一式内容。除初联代数外，还考查一式内容（体现在填空题中）。其他板块（几何、数论、组合）基本要求初联水平，几何稍难于初联。
录取与分数线 分数线按人头划定，不特别重要。实际操作中：擅长板块做一两道大题 + 填空题做两三道 + 其余题目写点过程猜答案，即可达到录取线。
板块 知识范围 题量 备注
代数 一试 + 初联代数 填空6题 + 大题2题 唯一含填空题的板块，进度要求最高
几何 初联几何（稍难） 大题2题 涉及位似、切割线、圆幂等
数论 初联数论 大题2题 侧重数感与配对思想
组合 初联组合 大题2题 概率博弈 + 组合计数
出题理念 注重思维考察而非技巧堆砌。不刻意使用均值、柯西、排序等高级不等式，而是考察学生能否大胆放缩、能否发现隐藏结构。思维能力是学生自己的，技巧可以在后续学习中习得。
SECTION 02

填空题解析（一试·代数）

六道填空题，考察一试知识，涵盖三角函数、消元、二进制、复数、博弈概率、组合计数

1三角函数单调性
难度
基础

考点：三角函数的严格单调递增区间，需分类讨论。

1

求导转化

对原函数求导，得到含 2ω·cos(2ωx + π/6) 的导函数。将原函数的单调递增问题转化为导函数 ≥ 0 的条件。

2

分 ω 正负讨论

因导函数前有系数 2ω，需分 ω < 0 和 ω > 0 两种情况。利用 cos 函数的正负区间建立不等式。

3

区间长度约束

快速方法：计算区间长度 −2ωπ/3，要求其 ≤ π，由此快速锁定 ω 的范围。

4

合并结果

ω < 0 时得 [−5/3, −1]；ω > 0 时得 (0, 1/3] ∪ [2, 7/3]。注意 ω = 0 不可取（否则退化为常函数，不满足"严格"单调递增）。

易错点 计算量较大，分类讨论易出错。尤其注意 ω = 0 为开区间，不可遗漏。
2消元与二次函数
难度
基础

考点：多元函数消元思想，将三元问题降为二次函数最值。

关键突破口：观察到变量 c 仅出现一次项（无平方项），因此直接用 c = 3 − a − b 代入消元。
  • 消去 c 后，原式化为关于 a + b 的二次函数
  • 令 t = a + b，得到 −3t² + 15t
  • 配方：最大值在 t = 5/2 时取到，值为 75/4
设计意图 本题不依赖复杂不等式（均值、柯西等），仅需观察结构特征并实施消元，是纯粹的思维导向题。
3多项式与二进制表示
难度
中等

考点：多项式展开中特定次数项系数的确定，本质是二进制分解。

核心观察：展开式中每个括号的次数均为 2 的幂（1, 2, 4, 8, …, 1024）。要得到 x 的 2025 次方，需选择若干个不同的 2 的幂之和恰好等于 2025——这等价于 2025 的二进制表示，具有唯一性。
1

二进制分解

2025 = 2048 − 22 = (211 − 1) − (2 + 4 + 16)，即从全 1 的 11 位中去掉第 1、2、4 次幂对应的括号。

2

确定系数

未被选中（即取常数项而非 x 项）的括号为：第 1 个（常数 2）、第 2 个（常数 4）、第 4 个（常数 16）。系数 = 2 × 4 × 16 = 27。

策略要点 一式代数中常见的"包装"技巧——将本质（二进制）隐藏在多项式外壳之下。需要通过积累识别背后的真实结构。
4复数与均值不等式
难度
中等偏上

考点：复数三角形式、实虚部分离、均值不等式。未学复数者需跳过。

关键步骤——平方相加：将两个等式（实部为零、虚部为四）分别平方后相加。cos²+sin²=1 自动消去角度，交叉项恰好为 cos(θ₁−θ₂)，从而将复数问题转化为纯实数问题。
  • 由平方相加得：cos(θ₁−θ₂) = [16 − (r₁² + r₂²)] / (2r₁r₂)
  • 已知 r₁r₂ = 12，对 r₁² + r₂² 施用均值不等式：r₁² + r₂² ≥ 2r₁r₂ = 24
  • 代入得 cos(θ₁−θ₂) ≤ (16 − 24) / 24 = −1/3
进度门槛 未学过复数的学生无法完成三角形式的设定，在翻译阶段即会受阻。这属于对学习进度的考察。
5三个快枪手博弈
难度
中等偏上

考点：概率与博弈论结合，分类讨论策略。

问题设定 三人（甲·乙·丙）按序开枪，甲命中率 10%，乙命中率 100%，丙命中率为 p（1/10 < p < 1）。乙、丙均为完全理性。已知"打或不打，甲的胜率相同"，求 p。

策略分析

  • 甲若射击，应打乙（最强威胁）
  • 乙理性选择先杀丙（威胁更大者）
  • 甲"放空枪"等价于让乙先清除丙

胜率计算

  • 无论甲是否开枪（或打中/未打中），其总胜率均为 10%
  • 关键条件：丙先开枪 vs 甲先开枪时，胜率之比为 9:1
  • 即 p : [(1−p) × 0.1] = 9 : 1
答案 解方程 p = 9(1−p) × 0.1，得 p = 9/19。
反直觉结论 在某些命中率配置下，"放空枪"（让更强者先清除第三方）是最优策略。这体现了博弈论中先发优势的微妙之处。
6二次函数与组合计数
难度
中等

考点：二次函数顶点位置约束下的组合计数，考察分类讨论能力。

1

确定 c

函数过点 (0, 2)，直接得 c = 2。

2

顶点坐标

顶点为 (−b/2a, (8a−b²)/4a)，需在第一或第三象限。

3

分类枚举

从 {−3, −2, −1, 1, 2, 3, 5} 中选取两两不同的 a、b，逐一枚举满足条件的组合。

  • 顶点在第一象限：a、b 反号，且 (8a−b²)/4a > 0 → 共 11 种
  • 顶点在第三象限：a、b 同号，且 (8a−b²)/4a < 0 → 共 1 种
  • 总计 12 种
填空题小结 前四题为经典一式代数（三角函数、消元、二进制、复数），后两题为概率与组合计数。整体强调分类讨论思维和对隐藏结构的识别能力。
SECTION 03

代数大题解析

两道大题，均为 12 分，考察放缩思维与非对称分析

1最大值的最小值——大胆放缩
难度
初联难度

题目特征：七个非负变量之和为 2027，A 为五个分式的最大值，求 A 的最小值。

核心思路——"抛弃完美主义"： 传统做法要求五个分式的系数恰好组合成 k(x₁+…+x₇) 的形式，但本题中五个分式完全不对称，无法精确配凑。解题关键是大胆允许多余项——多出来的变量直接令其为零即可。
1

选三个分式相加

挑选三个分式使其覆盖 x₁ 至 x₇ 的所有变量。得到 3A ≥ x₁ + 2x₂ + 2x₃ + x₄ + 2x₅ + x₆ + x₇。

2

放掉多余项

多出的 x₂、x₃、x₅ 令其为零（非负约束允许），直接得 3A ≥ x₁+…+x₇ = 2027。

3

验证取等

令 x₂=x₃=x₅=x₆=0，x₁=x₄=x₇=2027/3，五个分式均为 2027/3，等号成立。

答案 A 的最小值为 2027/3。
设计意图 一道 12 分大题，难度仅相当于初联水平。考察的不是高深不等式，而是能否突破"必须完美配凑"的思维定式。只要足够大胆，三个不等式相加即可解决问题。
2非对称分析——定向放缩
难度
较难

题目特征：x+y+z=0，x²+y²+z²=1，x≥y≥z。求 ab 的最大值，其中 a 为最大变量，b 为最小变量（即 a=x，b=z）。

核心思路——定向放缩：条件完全对称，但目标 xz 不对称。关键在于利用排序和 x+y+z=0 的约束，将所有项定向放缩为 xz 的函数。
1

不妨设 y ≥ 0

若 y < 0，将 x,y,z 全部取负，条件不变且 xz 不变，故不失一般性。

2

放缩 x²+y² 和 z²

x²+y² ≤ x²+xy = x(x+y) = −xz；z² = (x+y)² = −xz − yz ≤ −xz − xz = −2xz。

3

合并得界

1 = x²+y²+z² ≤ −xz − 2xz = −3xz，即 xz ≤ −1/3。

答案 xz 的最大值为 −1/3。取等条件：x = y = √6/6，z = −√6/3。
与第一题的共性 两道代数大题均不使用均值、柯西等高级工具，仅依赖初联水平的放缩技巧。考察核心是思维方向的正确性：第一题需要"大胆"，第二题需要"定向"。
SECTION 04

几何大题解析

两道大题，均 12 分，涉及位似、切割线定理、圆幂、对称构造等

1两圆内切与二倍角构造
难度
中等

第一问：三点共线

  • G 为两圆切点，即位似中心
  • A 为大圆"北极点"，K 为小圆"北极点"——地位相同
  • 由位似关系，A、K、G 三点共线
考试技巧 若第一问看不出位似，可默认结论正确，直接利用其推导第二问，仍可得分。

第二问：二倍角

  • 目标：∠BAD = 2∠CDF
  • 策略：构造 2∠CDF → 证明角相等
  • 将 F 关于 CD 对称得 F'，则 ∠FDF' = 2∠CDF
  • 转化为证明 △FDF' ∼ △FAD（两等腰三角形共享底角）
  • 只需证 AD = AF
AD = AF 的证明链： AD² = AC·AB（射影定理）→ B,C,K,J 四点共圆（两个直角）→ AC·AB = AK·AJ（切割线定理，需 A,K,J 共线）→ AK·AJ = AF²（A 对小圆的圆幂）→ AD = AF。
2外心转化与对称构造
难度
最难

题目特征：三角形外接圆、内切圆、平行切线、直角条件。证明 AD = DI。

本卷最难的题目之一 AD = DI 无法通过直接导角或导边证明。解题需要一次"异想天开"的对称构造。
1

自然辅助线

画外接圆；延长 AI 交外接圆于 M（弧 BC 中点，鸡爪定理）；连 OM（⊥BC）。利用切线平行于 BC 及 ∠OID = 90° 进行导角。

2

关键对称（核心难点）

将 I 关于过 D 的垂线（即过切点与 I 的连线）对称得 I'。将 DI = DA 的两线段相等问题转化为 D 为 △AII' 外心的三线段相等：DI = DI' = DA。
3

圆心角 = 2 倍圆周角

D 已在 II' 的中垂线上。若能证明 ∠IDI' = 2∠IAI'，则 D 为外心。而 ∠IDI' = 2∠PDI = 2∠MOI，故只需证 ∠MOI = ∠IAI'。

4

三角形相似 + 经典结论

△AII' ∼ △OMI，需验证 AI·IM = OM·II'。左边 = AI·IM = 2Rr（I 对外接圆的幂），右边 = R·2r = 2Rr，恒等式成立。证毕。

思路特征 本题的核心难度在于"对称构造"——将两线段相等转化为三线段相等（外心），这一思路并不自然。一旦想到，后续相似与圆幂结论均为熟知结论。
SECTION 05

数论大题解析

两道大题，考察余数配对思想与约数结构分析

1配对思想求 aₖ

定义：固定 k，aₖ = #{m : 1 ≤ m ≤ 2025, km mod 2025 > m}。

核心观察——配对：m 与 2025−m 在模 2025 意义下互为相反数。乘 k 之后仍然相反。因此，若 km mod 2025 > m，则 k(2025−m) mod 2025 < 2025−m。两者中恰好一个满足条件。

排除特殊情况

  • 情况一：km ≡ 0 (mod 2025)，两边均不大于原值
  • 情况二：km ≡ m (mod 2025)，即 (k−1)m ≡ 0 (mod 2025)，两边均不大于原值

计算 aₖ

aₖ = (2024 − 情况一的个数 − 情况二的个数) / 2

排除 2025 自身（恒不计入），总数从 2024 开始。

第一问答案 a₆ = 1009。
2不同值的个数

需要统计 a₁ 到 a₂₀₂₅ 中有多少个不同的值。

简化条件：aₖ 的值取决于 d₁(k) + d₂(k) 的不同取值，其中 d₁(k) = gcd(k, 2025)，d₂(k) = gcd(k−1, 2025)。因 d₁、d₂ 分别是 k 与 k−1 的约数，必互素。2025 = 3⁴ × 5²，互素的约数对只有两种结构。
  • 类型一：其中一个为 1，另一个为 2025 的任意约数 → (4+1)(2+1) × 2 = 30 种（去重后）
  • 类型二：一个只含 3 的幂（3¹ 至 3⁴），另一个只含 5 的幂（5¹ 至 5²）→ 4 × 2 = 8 种
  • 排除重叠后共 23 种不同取值
第二问答案 a₁ 到 a₂₀₂₅ 中有 23 个不同的值。
SECTION 06

组合大题解析

两道大题，考察概率博弈与组合计数

1概率博弈
说明 组合第一题（概率博弈）与填空题第五题（三个快枪手）属同一主题——概率与博弈论的结合。解题框架一致：分类讨论 → 分析各策略胜率 → 利用理性假设建立等式。
2组合计数与二次函数
说明 组合第二题（组合计数）与填空题第六题属同一主题——在二次函数约束下的组合计数。核心方法：确定固定参数 → 枚举满足条件的变量组合 → 分类求和。
组合板块特点 两道大题均需要较强的分类讨论能力和数感，答案书写较为直接，但对思维的"灵光一现"要求较高。
SECTION 07

做题策略与考试技巧

时间分配、取舍原则与应试心态

时间分配建议

  • 填空题（代数）：优先完成有把握的题目
  • 每个板块先做第一题（相对简单）
  • 难题可先跳过，最后处理

得分策略

  • 擅长板块做大题 + 填空做 2-3 道 + 其余写过程猜答案 → 基本可过线
  • 题难 ≠ 考试难：竞争环境下多拿分即可
  • 大题第一问不会时，默认其结论正确推导第二问
核心结论 本卷设计有意避免技巧堆砌（均值、柯西、排序等均未作为核心工具），所有题目均可在初联知识框架内完成。决定成败的是思维方向的选择：能否大胆放缩、能否发现隐藏结构、能否构造有效辅助线。
关于答案复核 有学生第三题答案为 17 而非 7，如因该 5 分差异未通过录取线，可申请复核。
